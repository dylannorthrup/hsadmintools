Update Ban List

This form will add or remove someone from being active on the Hearthstone Ban List that is consulted when checking match registrations. When using this form, you'll need to enter the following information:
  • The Battletag of the player (eg. DylanMcCheaty#12345, 恶棍#2121, 사기꾼#9999, etc.)
  • Your Admin name (eg. Dylan, Jeph, Tazwolf, etc.)
  • Select 'Add' or 'Remove' from the Action drop-down
  • The form password (which can be found in the normal places passwords are found)
  • Notes on why you are taking this action (not strictly necessary, but always a good thing to put in for tracking purposes)

Use this information for good, not for evil.
